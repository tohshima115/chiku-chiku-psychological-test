# チクチク心理テスト 仕様書

## コンセプト

エセ心理学の分類は「どの結果でも嬉しい」ように言い換えられている。このサイトは、同じ分類を**身も蓋もない言い方（チクチク言葉）**で伝えることで、その欺瞞を笑いに変える診断テスト。

ページ上のトグルで「チクチク言葉（本音）」と「ぽかぽか言葉（従来の優しい表現）」を切り替えられ、**同じ診断結果のギャップ**を楽しめる。

## サイト構成

### ページ一覧

| パス | 内容 | レンダリング |
|---|---|---|
| `/` | トップページ。コンセプト説明 + 導線 | 静的 (SSG) |
| `/onboarding` | 血液型オンボーディング体験 | 静的 + Reactアイランド |
| `/test` | メイン診断（10〜15問） | 静的 + Reactアイランド |
| `/result/[type]` | 結果表示 + SNSシェア | SSR（OGPメタタグ動的生成） |
| `/og/[type].png` | OGP画像APIエンドポイント | SSR（satori画像生成） |

### ユーザーフロー

```
トップページ
  ├─「まずは血液型で体験」→ /onboarding
  │   └─ 血液型選択 → ぽかぽか結果表示
  │       └─「本音バージョンを見る？」→ チクチク版に切替
  │           └─「本編の診断へ」→ /test
  └─「本編の診断を始める」→ /test
        └─ 10〜15問回答 → スコア集計 → /result/[type]
              └─ 結果表示（トグルで切替）+ SNSシェア
```

## 診断体系

### 1. オンボーディング: 血液型診断

血液型を選ぶだけで、チクチク/ぽかぽかの切り替え体験を提供する導入コンテンツ。

| 血液型 | ぽかぽか名 | ぽかぽか説明 | チクチク名 | チクチク説明 |
|---|---|---|---|---|
| A型 | 几帳面で真面目なしっかり者 | 計画的に物事を進め、周囲からの信頼も厚い | 神経質 | 細かいことが気になって仕方ない。周りもちょっと疲れてる |
| B型 | マイペースで自由な開拓者 | 自分の信じた道を突き進む情熱家 | 自分勝手 | 周りの都合は基本見えてない。本人だけ楽しそう |
| O型 | おおらかなリーダー気質 | 懐が深く、人をまとめる力がある | 雑 | 細かいことは気にしない、というか気づいてない |
| AB型 | 天才肌のミステリアス | 独自の感性で周囲を魅了する不思議な存在 | 何考えてるかわからない | 本人もたぶんわかってない |

### 2. メイン診断: オリジナルタイプ

#### スコアリング軸（4軸）

| 軸 | 英語キー | 低スコア側 | 高スコア側 |
|---|---|---|---|
| 社交性 | `sociability` | 内向的 | 外向的 |
| 計画性 | `planning` | 即興的 | 計画的 |
| 感受性 | `sensitivity` | 鈍感 | 敏感 |
| 主張性 | `assertiveness` | 受動的 | 主張的 |

#### 結果タイプ（8タイプ）

各軸を「高/低」で2分し、特徴的な4軸の組み合わせパターンから8タイプに分類する。

| ID | ぽかぽか名 | チクチク名 | 軸の傾向 | ぽかぽか説明 | チクチク説明 |
|---|---|---|---|---|---|
| `philosopher` | 思慮深い哲学者 | 考えすぎて動けない人 | 内向・即興・敏感・受動 | 深い洞察力を持ち、物事の本質を見抜く | ずっと考えてるけど何も始まらない。頭の中だけ忙しい |
| `commander` | 頼れる司令官 | 仕切りたがりの独裁者 | 外向・計画・鈍感・主張 | 明確なビジョンで人を導くリーダー | 人の話を聞かずに仕切り始める。本人は「リーダーシップ」と呼んでる |
| `diplomat` | 心優しい調停者 | 八方美人 | 外向・即興・敏感・受動 | 誰とでも打ち解け、場の空気を和ませる存在 | みんなにいい顔して結局誰の味方でもない |
| `strategist` | 冷静な戦略家 | 冷たい計算機 | 内向・計画・鈍感・主張 | 論理的に最善の道を選び、着実に成果を出す | 効率のことしか考えてないから一緒にいて楽しくない |
| `adventurer` | 自由な冒険家 | 飽き性の逃げ足職人 | 外向・即興・鈍感・主張 | 好奇心旺盛でどんな環境でも楽しめるチャレンジャー | 新しいことに飛びつくけど続いた試しがない |
| `guardian` | 誠実な守護者 | 心配性のお節介 | 内向・計画・敏感・受動 | 大切な人を静かに見守り、支える縁の下の力持ち | 頼まれてないのに心配して、頼まれてないアドバイスをする |
| `performer` | 輝くエンターテイナー | 注目されたいだけの人 | 外向・即興・敏感・主張 | カリスマ性があり、周囲を明るく照らす太陽のような人 | 常に主役でいたい。会話のハイジャックが得意 |
| `craftsman` | 職人気質の完璧主義者 | うるさい細かい人 | 内向・計画・鈍感・受動 | こだわりを持って質の高い成果を生み出す匠 | 自分ルールが多すぎて周りがついていけない |

#### アドバイス（各タイプ）

各タイプにチクチク/ぽかぽか両方のアドバイスを用意する。

例（`philosopher`）:
- **ぽかぽか**: 「あなたの深い思考は大きな武器。時には直感を信じて一歩踏み出すと、新しい世界が広がります」
- **チクチク**: 「考えるのはもういいから動け。完璧なタイミングは一生来ない」

## 質問設計

### 形式
- **問数**: 12問
- **選択肢**: 各問4択
- **スコアリング**: 各選択肢が4軸それぞれにスコア（-2〜+2）を加算
- **質問文**: チクチク/ぽかぽか両バージョン

### 質問一覧

#### Q1
- **ぽかぽか**: 「休日の過ごし方で理想に近いのは？」
- **チクチク**: 「休日、実際のところ何してる？」
- 選択肢:
  1. 友人と出かける（sociability+2, assertiveness+1）
  2. 家で一人でのんびり（sociability-2, planning-1）
  3. 趣味に没頭する（planning+1, sensitivity+1）
  4. 予定を詰め込む（planning+2, assertiveness+1）

#### Q2
- **ぽかぽか**: 「グループ作業で自然と担う役割は？」
- **チクチク**: 「グループ作業、正直どうしてる？」
- 選択肢:
  1. みんなの意見をまとめる（sociability+1, assertiveness+2）
  2. 黙々と自分の担当をこなす（sociability-1, planning+2）
  3. 雰囲気づくりを担当（sociability+2, sensitivity+1）
  4. 正直あまり関わりたくない（sociability-2, assertiveness-1）

#### Q3
- **ぽかぽか**: 「計画が変更になったとき、あなたは？」
- **チクチク**: 「予定が急に変わったら？」
- 選択肢:
  1. 柔軟に対応できる（planning-2, sociability+1）
  2. 少しストレスを感じる（planning+1, sensitivity+2）
  3. すぐ新しい計画を立てる（planning+2, assertiveness+1）
  4. 別にどうでもいい（sensitivity-2, planning-1）

#### Q4
- **ぽかぽか**: 「友人が落ち込んでいるとき、どうしますか？」
- **チクチク**: 「友達が病んでLINEしてきたら？」
- 選択肢:
  1. じっくり話を聞く（sensitivity+2, sociability+1）
  2. 具体的な解決策を提案する（assertiveness+2, sensitivity-1）
  3. 気分転換に誘い出す（sociability+2, assertiveness+1）
  4. そっとしておく（sensitivity+1, sociability-2）

#### Q5
- **ぽかぽか**: 「自分の意見が少数派だったとき、どうしますか？」
- **チクチク**: 「みんなと意見違ったらどうする？」
- 選択肢:
  1. 自分の意見を主張する（assertiveness+2, sociability+1）
  2. 多数派に合わせる（assertiveness-2, sensitivity+1）
  3. 黙って様子を見る（assertiveness-1, sociability-1）
  4. 別の切り口で提案する（assertiveness+1, planning+1）

#### Q6
- **ぽかぽか**: 「新しい環境に入ったとき、最初にすることは？」
- **チクチク**: 「知らない場所に放り込まれたら？」
- 選択肢:
  1. 積極的に話しかける（sociability+2, assertiveness+2）
  2. まず観察してから動く（planning+1, sensitivity+1）
  3. 一人でも居心地の良い場所を探す（sociability-2, sensitivity+1）
  4. 誰かが話しかけてくれるのを待つ（assertiveness-2, sociability-1）

#### Q7
- **ぽかぽか**: 「仕事や勉強のスタイルは？」
- **チクチク**: 「締め切り前の自分を正直に言うと？」
- 選択肢:
  1. コツコツ計画通りに進める（planning+2, sensitivity-1）
  2. 締め切り直前に本気出す（planning-2, assertiveness+1）
  3. 完璧になるまでやり直す（sensitivity+2, planning+1）
  4. 最低限で終わらせる（sensitivity-2, assertiveness-1）

#### Q8
- **ぽかぽか**: 「あなたが大切にしている価値観は？」
- **チクチク**: 「結局、何が一番大事？」
- 選択肢:
  1. 人とのつながり（sociability+2, sensitivity+1）
  2. 自由と独立（sociability-1, assertiveness+2）
  3. 安定と安心（planning+2, sensitivity+1）
  4. 成果と達成感（assertiveness+2, planning+1）

#### Q9
- **ぽかぽか**: 「ストレスを感じたときの対処法は？」
- **チクチク**: 「イライラしたとき、本当はどうしてる？」
- 選択肢:
  1. 誰かに話を聞いてもらう（sociability+2, sensitivity+1）
  2. 一人で静かに過ごす（sociability-2, sensitivity+1）
  3. 体を動かしたり行動で発散（assertiveness+1, planning-1）
  4. 原因を分析して対策を考える（planning+2, assertiveness+1）

#### Q10
- **ぽかぽか**: 「チームの中であなたが最も貢献できることは？」
- **チクチク**: 「チームで唯一マシなところは？」
- 選択肢:
  1. 場の雰囲気を良くすること（sociability+2, sensitivity+1）
  2. 計画を立てて管理すること（planning+2, assertiveness+1）
  3. 新しいアイデアを出すこと（planning-1, assertiveness+1）
  4. 黙って確実にこなすこと（sociability-1, planning+1）

#### Q11
- **ぽかぽか**: 「初対面の人との会話で心がけていることは？」
- **チクチク**: 「初対面の人と話すとき、本音は？」
- 選択肢:
  1. 笑顔で積極的に話す（sociability+2, assertiveness+1）
  2. 相手の話をよく聞く（sensitivity+2, assertiveness-1）
  3. 共通点を探す（sociability+1, planning+1）
  4. 早く終わらないかなと思っている（sociability-2, sensitivity-1）

#### Q12
- **ぽかぽか**: 「人生で最も成長を感じる瞬間は？」
- **チクチク**: 「ぶっちゃけ一番テンション上がるのは？」
- 選択肢:
  1. 人に感謝されたとき（sensitivity+2, sociability+1）
  2. 目標を達成したとき（assertiveness+2, planning+1）
  3. 新しい発見をしたとき（planning-1, sensitivity+1）
  4. 自分のペースで過ごせているとき（sociability-2, planning+1）

## スコアリングロジック

1. 全12問の回答から4軸のスコアを合計
2. 各軸を「高（0以上）/低（0未満）」で2分
3. 4軸の高低パターンからタイプを決定

| sociability | planning | sensitivity | assertiveness | タイプ |
|---|---|---|---|---|
| 低 | 低 | 高 | 低 | `philosopher` |
| 高 | 高 | 低 | 高 | `commander` |
| 高 | 低 | 高 | 低 | `diplomat` |
| 低 | 高 | 低 | 高 | `strategist` |
| 高 | 低 | 低 | 高 | `adventurer` |
| 低 | 高 | 高 | 低 | `guardian` |
| 高 | 低 | 高 | 高 | `performer` |
| 低 | 高 | 低 | 低 | `craftsman` |

※ 完全一致しない場合は、各タイプとのスコアパターンの距離が最も近いタイプを選択。

## チクチク/ぽかぽか切り替えUI

### デザインコンセプト

| 要素 | ぽかぽかモード | チクチクモード |
|---|---|---|
| 配色 | 暖色系（ピンク、オレンジ） | 寒色/モノクロ系（グレー、ダークブルー） |
| フォントの雰囲気 | 柔らかい | シャープ |
| トグルラベル | 🌸 ぽかぽか | 🌵 チクチク |
| 切り替え演出 | テキストのフェードイン/アウト + 背景色のトランジション |

### 切り替えの挙動

- **オンボーディング**: デフォルトはぽかぽか。結果表示後に「本音バージョンを見る？」ボタンで初めてチクチクに切替。ギャップのインパクトを最大化。
- **メイン診断**: トグルスイッチが常時表示。いつでも切り替え可能。質問文・選択肢・結果すべてが即座に切り替わる。
- **結果ページ**: URLパラメータ `?mode=chiku` / `?mode=poka` でモードを保持。SNSシェア時にモード情報も共有。

### OGP/SNSシェア

- 各結果ページで `<meta property="og:image">` を動的に設定
- OGP画像には結果タイプ名（チクチク/ぽかぽか両方）を表示
- Xシェアテキスト: 「チクチク心理テストの結果は【○○】でした（本音：△△）」
